## cesium相机
#### 通过相机设计目的地：默认跳转过去的地方
```js
const position = Cesium.Cartesian3.fromDegrees(100, 100, 20)
 viewer.camera.setView({
     destination: position,
     orientation: {
         //左右
         heading: Cesium.Math.toRadians(0),
         //上下
         pitch: Cesium.Math.toRadians(-90),
         //旋转
         roll: Cesium.Math.toRadians(0)
     }
 })
```
